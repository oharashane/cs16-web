About CS1.6 Web

CS1.6 Web is a browser-based Counter-Strike 1.6 client using the yohimik WebAssembly engine with an enhanced Go WebRTC server.

Features

  • Browser-based Counter-Strike 1.6 gameplay
  • Multi-server support with automatic discovery
  • WebRTC transport for browser networking
  • Enhanced Go server with integrated UDP relay
  • Dynamic port management for multiple CS servers

Built With

  • yohimik WebAssembly engine
  • Xash3D engine
  • cs16.css UI library
